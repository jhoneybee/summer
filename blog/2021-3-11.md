2021 年, 想打造一款符合Admin后端管理的UI解决方案. 

初步构想UI库的组件信息如下

- Button 按钮
- Input 行文本输入
- Radio   单选框
- Checkbox 多选框
- DatePicker 日期选择框
- Select 选择器
- Message 全局提示
- Password 密码输入框
- Tabs 标签页
- Tree 树形控件
- Table 表格组件


## 设计思路如下


- 让开发人员易用, API简洁易懂
- 高性能组件, 所有地方需要考虑大数据加载情况
